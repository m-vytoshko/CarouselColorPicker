#### Implementation 
Add it in your root build.gradle at the end of repositories:
```
allprojects {
    repositories {
	...
        maven { url 'https://jitpack.io' }
    }
}
```
Add the dependency
```
implementation("com.github.m-vytoshko:CarouselColorPicker:latestVersion")
```
